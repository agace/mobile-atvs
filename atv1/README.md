## Objetivo

Configurar o ambiente de desenvolvimento Dart e executar um programa simples que imprime "Hello World" no terminal, demonstrando o uso de Git e GitHub para documentar e versionar o projeto.

## Como executar

1. Certifique-se de ter o [Dart SDK](https://dart.dev/get-dart) instalado
2. Navegue ate o diretorio do projeto:

```bash
cd atv1
```

3. Execute o programa:

```bash
dart run
```

## Saida esperada

```text
Hello World
```
