# 🎬 Desafios da Aula: Navegação e Roteamento com AutoRoute

## Contexto

Nesta aula, evoluímos o aplicativo de **Catálogo de Filmes** para implementar navegação entre telas usando **AutoRoute**. Atualmente, o código utiliza navegação imperativa com `Navigator.push()`. Seu desafio é refatorar para usar AutoRoute.

---

## 📋 Desafio 1: Configuração do AutoRoute

Implemente a configuração de AutoRoute com dois `AutoRoute` para as telas do catálogo e detalhes.

---

## 📱 Desafio 2: Integração no App

Integre AutoRoute no `MaterialApp` usando `routerConfig`.

---

## 🎯 Desafio 3: Navegação Moderna

Refatore o `onTap` para usar AutoRoute em vez de `Navigator.push()`.
